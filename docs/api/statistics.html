총계 처리 클래스¶

DIL.statistics.Aggregation(datas: pandas.core.frame.DataFrame)¶
통계 기술 중 총계 기술(평균값, 최댓값, 최솟값, 최빈값, 중간값)을 구현한 클래스
모든 메소드는 생성자에 원본 데이터를 인자 값으로 넣으면 원본 데이터를 수정한다.
매개변수

datas (pandas.DataFrame) – 총계 기술을 적용할 DataFrame 지정
