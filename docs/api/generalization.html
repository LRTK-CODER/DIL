일반화 기술¶

로컬 일반화 기술 & 문자데이터 범주화 기술¶

일반화 기술 클래스¶

DIL.Generalization(datas: pandas.core.frame.DataFrame)¶
일반화 기술 중 로컬 일반화, 문자데이터 범주화를 구현한 클래스
모든 메소드는 생성자에 원본 데이터를 인자 값으로 넣으면 원본 데이터를 수정한다.
매개변수

datas (pandas.DataFrame) – 로컬 일반화 기술과 문자데이터 범주화 기술을 적용할 DataFrame 지정

로컬 일반화 메소드¶

DIL.Generalization.local()¶

일반화 기술 중 로컬 일반화를 수행하는 함수

매개변수

  • column (str) – 로컬 일반화를 적용할 컬럼

  • currentIndexList (list) – 로컬 일반화를 적용할 Index 범위

반환값

기술 적용 성공 시 True 리턴

반환 형식

bool

문자데이터 범주화 메소드¶

DIL.Generalization.categorizion()¶

일반화 기술 중 문자데이터 범주화를 수행하는 함수

매개변수

  • column (str) – 문자데이터 범주화를 적용할 컬럼

  • replaceList (list) – 범주화로 치환될 문자 데이터 목록

  • category (str) – 범주화로 치환할 문자 데이터

반환값

기술 적용 성공 시 True 리턴

반환 형식

bool

라운딩 클래스¶

DIL.Rounding()¶

라운딩 기술(일반 라운딩, 랜덤 라운딩)을 구현한 클래스

반올림 메소드¶

DIL.Rounding.off()¶

라운딩 기술 중 반올림을 수행하는 메소드

매개변수

  • data (int) – 반올림을 수행할 숫자형 타입 데이터

  • seatNum (int) – 반올림을 수행할 숫자형 타입 데이터의 자리수 위치

반환값

반올림이 적용된 숫자형 타입 데이터를 리턴

반환 형식

int
