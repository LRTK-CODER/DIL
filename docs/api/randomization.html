잡음 추가 기술¶

잡음 추가 기술 클래스¶

DIL.randomization.Noise(datas: pandas.core.frame.DataFrame)¶
무작위화 기술 중 잡음추가 기술을 구현한 클래스
모든 메소드는 생성자에 원본 데이터를 인자 값으로 넣으면 원본 데이터를 수정한다.
매개변수

datas (pandas.DataFrame) – 잡음추가 기술을 적용할 DataFrame 지정

단일 컬럼 잡음 추가 메소드¶

DIL.randomization.Noise.add()¶

잡음추가 기술로 한가지 컬럼에만 잡음을 추가하는 메소드

매개변수

  • column (str) – 잡음 추가 기술을 적용할 컬럼

  • randomRange (list) – 잡음 추가 기술에 사용될 잡음의 범위

반환값

기술 적용 성공 시 True 리턴

반환 형식

bool

다중 컬럼 잡음 추가 메소드¶

DIL.randomization.Noise.multipleAdd()¶

잡음추가 기술로 여러개의 컬럼에 잡음을 추가하는 메소드

매개변수

  • columns (list) – 잡음 추가 기술을 적용할 컬럼들

  • randomRange (list) – 잡음 추가 기술에 사용될 잡음의 범위

반환값

기술 적용 성공 시 True 리턴

반환 형식

bool

순열(재배열) 기술 클래스¶

DIL.randomization.Permutation(datas: pandas.core.frame.DataFrame)¶
무작위화 기술 중 순열(재정렬) 기술을 구현한 클래스
모든 메소드는 생성자에 원본 데이터를 인자 값으로 넣으면 원본 데이터를 수정한다.
매개변수

datas (pandas.DataFrame) – 순열 기술을 적용할 DataFrame 지정

DIL.randomization.Permutation.all()¶

순열을 구현한 메소드로 한가지 컬럼의 모든 데이터의 순서를 재정렬하는 메소드

매개변수

column (str) – 순열 기술을 적용할 컬럼

반환값

기술 적용 성공 시 True 리턴

반환 형식

bool
